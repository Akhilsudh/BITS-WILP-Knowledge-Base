## Transformation of circle under matrix operation
![[Pasted image 20210912110427.png]]
```matlab
A = [1 2; 3 4]
%{
A =

   1   2
   3   4

}%
x1 = [1 2]'
%{
x1 =

   1
   2

}%
x2 = [-2 1]'
%{
x2 =

  -2
   1

}%
y1 = A * x1
%{
y1 =

    5
   11

}%
y2 = A * x2
%{
y2 =

   0
  -2

}%
```
